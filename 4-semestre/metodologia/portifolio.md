##  <p align="center">Meus Projetos</p>

### Morpheus 

<p align="justify">No terceiro semestre do curso eu e minha equipe Morpheus desenvolvemos um projeto para a empresa GSW . O desafio proposto para o projeto foi o desenvolvimento de uma aplicação web que realizasse a coleta e armazenamento de notícias estratégicas de diferentes portais de notícias e fontes de dados via API. Além disso, o sistema oferece funcionalidades para tratar regionalismos e sinônimos nas tags cadastradas, facilitando buscas e análises futuras. </p>

#### Contribuições Pessoais 

<p align="justify">Atuei diretamente no desenvolvimento do front-end da aplicação. As interfaces foram construídas com Vue.js 3 e integrando-se com a API backend para persistência dos dados. Todas as operações de cadastro, edição, exclusão e listagem foram implementadas com axios, garantindo a persistência e sincronia dos dados entre o front-end e o banco de dados.
